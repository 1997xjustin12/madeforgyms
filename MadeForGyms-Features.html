Section 02

Member Management

Register, search, view, and manage every member from one screen. Full profile support including photos, body metrics, and membership history.

➕

Member Registration

Add members with name, contact, photo, birthdate, height, weight, and membership plan in one form.

🔍

Instantly search by name or number. Filter by Active, Expiring, or Expired status with live counts.

Section 03

Membership Plans & Payments

Flexible plan options, promo support, GCash integration, and advance payment tracking — all built in.

Available Plan Types
  • Monthly — 30-day rolling membership
  • Quarterly — 90-day membership
  • Semi-Annual — 180-day membership
  • Annual — 365-day membership
  • Student — discounted 30-day plan (requires valid school ID)
  • Custom Promos — create unlimited special promotions with custom duration and price

Payment Workflows

💵

Quick Renew (Admin)

Admins can renew any member instantly — select plan, choose Cash or GCash, confirm. Membership extends immediately.

📲

GCash Self-Renewal

Members upload their GCash receipt from their phone. Admin gets notified and approves with one tap.

🗓️

Advance Payments

Accept partial or full payments in advance. The system queues them and applies to the next renewal automatically.

🔗

Review Payment Link

One-tap review link sent via Telegram. Admin can approve or reject renewal requests from anywhere — no login needed.

  • Renewal request queue — all pending GCash renewals in one approval screen with receipt viewing
  • Batch approve/reject — process multiple renewal requests at once
  • Auto end-date calculation — system automatically sets the new expiry date on approval
  • Advance payment carry-forward — excess balance rolls over to next renewal period
  • Payment method tracking — records whether payment was Cash or GCash for each transaction

Section 06

Workout Plans

Automatically generate personalized weekly workout plans based on each member's BMI and fitness level. Fully customizable by admin or coach.

How Auto-Generation Works

When a workout plan is created, the system uses the member's BMI category (Underweight, Normal, Overweight, Obese) and their fitness level (Beginner, Intermediate, Advanced) to pick the right template — a full 7-day weekly program with appropriate exercises pre-filled.

🤖

BMI-Based Templates

12 pre-built templates covering all BMI categories × fitness levels. Right program for every body type.

✏️

Fully Editable

Add, remove, or modify any exercise per day. Change sets, reps, rest time, and muscle group focus.

🎬

Exercise Media

Upload a demo video and/or image for any exercise. Members see the media directly in their workout view.

📆

7-Day Weekly Structure

Plans are organized by day (Mon–Sun) with rest days clearly marked. Members see today's day highlighted.

  • Day tabs — navigate between each day of the week to view or edit that day's exercises
  • Exercise details — each exercise shows name, sets, reps/duration, rest time, muscle group, and description
  • Muscle group labels — Chest, Back, Shoulders, Biceps, Triceps, Quads, Hamstrings, Glutes, Calves, Core, Full Body, Cardio
  • Reset to template — reset the entire plan back to the default template at any time
  • Member-side view — members see their plan in the portal with exercise media and set tracking
  • Fitness level selector — switch between Beginner, Intermediate, and Advanced to change template difficulty
Section 07

Coach & Trainer Management

Manage your coaching team, assign members to coaches, set up coaching packages, and give coaches their own secure login portal.

👨‍🏫

Coach Profiles

Each coach has a profile with name, specialty, photo, contact, email, bio, and active status toggle.

🔐

Coach Access Code

Coaches log in with a unique access code (no password needed). Admin sets and manages the code.

🗂️

Coach Dashboard

Coaches see all their assigned members, coaching subscription dates, and member details in their own portal.

📦

Coaching Packages

Create coaching subscription packages with custom name, price, and duration. Assign to members at registration.

  • Member–coach assignment — assign any coach to a member at registration or during profile editing
  • Coaching subscription dates — track coaching start and end dates separately from membership dates
  • Auto end-date calculation — select a package and the system calculates the end date automatically
  • Active/inactive toggle — deactivate a coach without deleting their record; they won't appear in the assignment list
  • Multiple packages — set up as many coaching packages as needed with different durations and prices
  • Coach portal separation — coaches only see their own members and cannot access admin settings or other members
Coaching Subscription Flow

When registering a member: select a coach → choose a package → start date is set → end date auto-calculates → saved with the member profile. Coaches can then view the member from their portal with the coaching timeline visible.

Section 08

Notifications & SMS Reminders

Keep members informed and reduce no-shows with automated SMS reminders and real-time Telegram alerts for admins.

💬

SMS Renewal Reminders

Send personalized SMS to individual members or all expiring members at once via Semaphore (Philippine SMS gateway).

📢

Bulk SMS

One-click bulk SMS to all members expiring within 5 days. Message includes name, days left, and GCash details.

📡

Telegram Admin Alerts

Instant Telegram notification when a member submits a GCash renewal — includes receipt and a one-tap approve link.

🔔

Expiry Alert on Dashboard

Admin dashboard highlights all members expiring within 5 days with a prominent alert section and send-SMS buttons.

  • Custom SMS message — each SMS is pre-filled with the member's name, days remaining, and gym's GCash number
  • Semaphore integration — uses Philippine SMS API with configurable sender name (up to 11 characters)
  • Telegram review link — secure 5-minute link lets admin approve renewals directly from Telegram without logging in
  • Automatic expiry reminders — system can be configured to auto-send SMS reminders at 9 AM daily
  • SMS from member list — send individual reminder SMS from any member's card in the members list
  • API key management — Semaphore API key and Telegram bot token stored securely in gym settings

Reports, Logs & Data Export

Full activity audit trail, data export to Excel, and backup/restore tools to keep your gym's data safe and accessible.

📊

Admin Dashboard Stats

Live counts for Total, Active, Expiring, and Expired members with active rate percentage at a glance.

📋

Activity Audit Log

Complete history of every action: member added, edited, deleted, renewed, SMS sent — with who did it and when.

  • Pagination for logs — large log histories load efficiently with page-by-page browsing
  • Logged actions include — registration, edits, deletion, renewal, SMS sent, coach assignment, and more

Section 11

User Roles & Permissions

MadeForGyms has three distinct user types — each with the right level of access for their role.

Three User Roles
  • Admin — full system access. Manages members, settings, payments, staff, and all data.
  • Staff — limited admin access. Can register members and record attendance, requires admin approval for changes.
  • Coach — portal-only access via access code. Sees only their assigned members and coaching details.
